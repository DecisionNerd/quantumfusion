# Sample Notebooks

This directory contains sample Jupyter notebooks that demonstrate how to use the `quantumfusion` package. These notebooks serve as educational resources and practical examples to help users get started with the library.

## Important Note

**These notebooks are NOT part of the installed package.** When you install `quantumfusion` from PyPI using `pip install quantumfusion`, these notebooks are not included in the installation. They are only available in the source repository on GitHub.

## Getting Started

To use these notebooks:

1. **Install the package** (if you haven't already):
   ```bash
   pip install quantumfusion
   ```

2. **Clone or download this repository** to access the notebooks:
   ```bash
   git clone https://github.com/yourusername/quantumfusion.git
   cd quantumfusion/notebooks
   ```

3. **Install Jupyter** (if needed):
   ```bash
   pip install jupyter
   ```

4. **Launch Jupyter**:
   ```bash
   jupyter notebook
   ```

## Notebook Contents

The notebooks in this directory provide:
- **Getting Started Guides**: Step-by-step tutorials for basic usage
- **Example Workflows**: Complete examples demonstrating common use cases
- **Advanced Techniques**: More sophisticated patterns and best practices
- **Integration Examples**: How to use quantumfusion with other Python libraries
